# EVORIX

Initial website for EVORIX, a futuristic annual interschool fest experience built with Next.js 15, TypeScript, Tailwind CSS, shadcn/ui conventions, Framer Motion, Prisma-ready structure, React Hook Form, Zod, and Lucide React.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Deployment Target

The project is structured for a future Render deployment. Database, authentication, dashboards, and Prisma schema implementation are intentionally deferred for the next phase.
